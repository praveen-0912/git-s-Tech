# 🚀 Projects

## 📈 Stock Market Prediction using Sentiment Analysis

### Technologies Used
- Python
- Flask
- XGBoost
- Pandas
- NumPy
- TextBlob
- Matplotlib

### Features

✔ Historical Stock Analysis

✔ News Sentiment Analysis

✔ Future Price Prediction

✔ Interactive Flask Web Application

### Description

This project predicts stock prices by combining historical stock data with real-time news sentiment analysis. Financial news is collected through Google News RSS and sentiment analysis is performed using TextBlob. XGBoost is used for forecasting future stock closing prices.
